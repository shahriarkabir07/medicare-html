# 💊 MediCare+ — Healthcare Landing Page

A colorful, responsive **Healthcare Landing Page** built with **pure HTML only** (inline styles, no external CSS or JavaScript frameworks).

## ✨ Features

| Section | Description |
|---|---|
| 🔵 Navbar | Logo + navigation links + Book Appointment button |
| 🏥 Hero | Headline, CTA buttons, animated stats (10K+ patients, 150+ doctors) |
| 💉 Services | 6 department cards — Cardiology, Neurology, Dentistry, Pediatrics, Orthopedics, Ophthalmology |
| 👨‍⚕️ Doctors | 4 doctor profile cards with specialty and star ratings |
| 📅 Appointment | Booking form with name, phone, department dropdown, date picker |
| ℹ️ About | Mission statement + 4 highlight badges |
| 📞 Contact | Address, Phone, Email, Hours |
| 🔻 Footer | Copyright info |

---

## 🎨 Design

- **Color Theme:** Colorful gradient (ocean blues, coral pinks, mint greens, warm yellows)
- **Font:** Georgia (system serif — no external font dependency)
- **Style:** Inline HTML styles only — zero external files
- **Emoji Icons:** Used throughout for visual richness without image assets

---

## 🚀 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/medicare-landing.git
   ```

2. **Open in browser**
   ```bash
   cd medicare-landing
   open index.html
   ```
   Or simply double-click `index.html` — it works instantly.

## 🛠️ Built With

- **HTML5** — structure and content
- **Inline CSS** — all styling done inside `style=""` attributes
- No JavaScript, no frameworks, no dependencies

4. Visit `https://YOUR_USERNAME.github.io/medicare-landing/`
